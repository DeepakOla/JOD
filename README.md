## Games

### Warp Lines
A kid-friendly abstract strategy board game inspired by checkers, played on a unique graph-based board.

**Features:**
- 🎮 Two game modes: Player vs Player and Player vs Computer
- 🎯 Strategic gameplay with mandatory capture rules
- 🎨 Beautiful visual design with smooth animations
- 📱 Works on both desktop and mobile devices
- 🚀 Single HTML file - no installation needed!

**How to Play:**
Simply open `warp-lines.html` in any modern web browser and start playing!

**Game Rules:**
- 2 players (Red vs Green), each with 13 pieces
- Move pieces along the graph edges to adjacent empty nodes
- Jump over opponent pieces to capture them
- If you can capture, you MUST capture
- Win by eliminating all opponent pieces or blocking their moves

For complete rules and documentation, see [WARP_LINES_PLAN.md](WARP_LINES_PLAN.md)
